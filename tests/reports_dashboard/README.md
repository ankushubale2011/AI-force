# Reports Dashboard Test

This test validates the Reports and Dashboard section of the application.

## Purpose
- Verify user login and navigation to Reports and Dashboard section
- Validate availability and correctness of current reports and dashboards
- Detect and test functionality of newly added reports and dashboards
- Report defects if new items are missing

## Framework & Language
- Framework: TestNG
- Language: Java (Spring Boot)

## Prerequisites
- Selenium WebDriver
- Chromedriver installed and path configured
- Application URL accessible

## Usage
Update the placeholders in `ReportsDashboardTest.java` for chromedriver path and application URL before running tests.
