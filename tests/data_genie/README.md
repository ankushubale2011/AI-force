# Data Genie Landing Page Test

This test validates the Data Genie landing page of the application.

## Purpose
- Verify navigation to Data Genie landing page
- Validate all links for correctness and functionality
- Ensure correct page redirection
- Verify correct loading of all images
- Check layout alignment
- Test responsiveness
- Check performance and console errors
- Validate accessibility using keyboard navigation
- Test browser and OS compatibility

## Framework & Language
- Framework: TestNG
- Language: Java (Spring Boot)

## Prerequisites
- Selenium WebDriver
- Chromedriver installed and path configured
- Application URL accessible

## Usage
Update the placeholders in `DataGenieLandingPageTest.java` for chromedriver path and application URL before running tests.
